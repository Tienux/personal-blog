[View the repo](https://github.com/Tienux/Minecraft_Linker_Bot_Rust_Edition)

## The project — what and why

This is my first public repository on GitHub — and deliberately open source
under the MIT license. The subject, a Discord bot for Minecraft, wasn't chosen
at random : it's familiar ground, which let me focus on what I actually wanted
to work on here.

The main goal was to practice structuring a project properly : setting up a CI
pipeline, organizing work with issues, following a clean GitHub workflow.
The bot itself is the excuse — building something real was the point.
