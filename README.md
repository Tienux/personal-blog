# Blog Portfolio

Portfolio et blog personnel construit avec Nuxt 4.

## Prérequis

- Node.js 20+ recommandé
- npm

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site démarre en local sur `http://localhost:3000`.

## Commandes utiles

```bash
npm run build
npm run preview
npm run lint
npm test
```

## Stack

- Nuxt 4
- Vue 3
- @nuxtjs/i18n
- Markdown parsé avec `marked` et nettoyé avec DOMPurify
